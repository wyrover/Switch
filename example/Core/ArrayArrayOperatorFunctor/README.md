# ArrayArrayOperatorFunctor
This code example shows how to use operator () to list the elements of an array.

## Output
```
The Array contains the following values:
(0, 0) The
(0, 1) quick
(0, 2) brown
(0, 3) fox
(0, 4) jumped
(1, 0) over
(1, 1) the
(1, 2) lazy
(1, 3) dog
(1, 4)
```
