# ArrayGetEnumerator
This code example shows how to use GetEnumerator to list the elements of an array.

```
The Array contains the following values:
[0] The
[1] quick
[2] brown
[3] fox
[4] jumped
[5] over
[6] the
[7] lazy
[8] dog
[9]
he quick brown fox jumps over three napping cats in the barn
```
